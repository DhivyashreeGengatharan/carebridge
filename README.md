# CareBridge  
### Post Discharge Navigator

CareBridge is a web-based healthcare support system designed to assist patients in managing their recovery after hospital discharge. It provides a seamless connection between patients and doctors for monitoring health, tracking medications, and managing appointments.

## Features

### Patient Features
- Track daily **vitals** (BP, Pulse, SpO2, Temperature, Weight, Pain)
- Manage **medications** and adherence
- View and manage **appointments**
- Maintain **daily recovery tasks**
- Raise **health concerns** to doctors
- Create **drawing notes** for visual tracking
- View recovery progress with charts and insights

### Doctor Features
- Monitor multiple **patients**
- View patient **vitals and reports**
- Manage **appointments**
- Respond to **patient concerns**
- Prescribe and update **medications**
- Access analytics and reports

## Tech Stack

- React.js (via CDN)
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage (for data persistence)
- Canvas API (for charts & drawing tools)

## Key Functionalities

- Interactive charts (Bar charts, Sparklines, Progress Rings)
- Local storage-based data management (offline-first)
- Authentication system (Patient & Doctor roles)
- Input validation for health data
- Drawing canvas for notes
- Responsive UI design
